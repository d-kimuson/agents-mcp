# @kimuson/agents-mcp

MCP server that provides unified interface for multiple AI agents (claude-code, codex, gemini-cli, etc.).

## Installation

```bash
npm install @kimuson/agents-mcp
```

## Development

```bash
# Install dependencies
pnpm install

# Type check
pnpm typecheck

# Lint
pnpm lint

# Fix lint issues
pnpm fix

# Build
pnpm build
```
